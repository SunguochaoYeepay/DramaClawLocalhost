# Fonts

## piko-countdown-pixel.woff

A digits-only (U+0030–U+0039) subset of **Ark Pixel / 方舟像素** (16px monospaced,
Latin), used to render the Piko mini-game countdown.

- Upstream: https://github.com/TakWolf/ark-pixel-font
- License: SIL Open Font License 1.1 — full text in `OFL.txt`
- Copyright (c) 2021, TakWolf (https://takwolf.com)

The subset was produced with `fonttools` (`pyftsubset --unicodes=U+0030-0039`).
Ark Pixel is OFL-licensed, so it may be freely bundled and redistributed as long as
`OFL.txt` ships alongside it. (It replaced an earlier iPix subset whose underlying
HZK16 bitmap source had unresolved copyright provenance.)
